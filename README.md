## 環境構築
1. ターミナルで以下のコマンドを実行
  ```sh
  git clone https://github.com/gizumo-education/webpack_practice.git && cd webpack_practice
  ```
2. `npm install`を実行しパッケージをインストール
3. `npm run watch`を実行
4. ブラウザが立ち上がり「Hello World.」が画面に表示されていることを確認
